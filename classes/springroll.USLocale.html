springroll.USLocale Class

Defined in: springroll.USLocale:44

USLocale is the standard keyboard layout for US computers. All standard keys are included. Keys to note:

  • 'OS' represents both the Windows key and the 'command' key, although 'command' is also available when the client is running MacOS.

  • 'meta' represents 'ctrl' on Windows and Linux, and 'command' on OSX. Use this when creating keyboard shortcuts in web apps.
